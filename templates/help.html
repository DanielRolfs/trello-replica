Learning Management System Project

How to use

  1. To create a new Task under go to "Add Task"
  2. After pressing the "CREATE TASK" button the Task will appear in the "Backlog" section.
  3. The assigned User can now take the task, by clicking on the Task in the "Backlog".
  4. The task is now copied from the "Backlog" section into the personal "Board".
  5. In the "Board" section, you can see the Progress of each Task.
  6. To bring the Task to the next level click on it.
If you need help or there are any problems contact us.
